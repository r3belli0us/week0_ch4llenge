# Climate Data Analysis - Week 0

## Overview
This project analyzes climate data from five African countries:
Ethiopia, Kenya, Sudan, Tanzania, and Nigeria.

The goal is to identify climate trends, seasonal patterns, and potential climate risks to support policy insights for COP32.

## Project Structure
- notebooks/ → contains all analysis notebooks
- data/ → ignored (datasets not uploaded)
- requirements.txt → dependencies used

## Analysis Performed
- Data cleaning and preprocessing
- Exploratory Data Analysis (EDA) for each country
- Cross-country comparison of:
  - Temperature (T2M)
  - Rainfall (PRECTOTCORR)
  - Extreme heat events
  - Humidity, wind speed, and variability

## How to Run
Open the notebooks in Google Colab and run all cells.

## Key Insight
The analysis highlights differences in climate patterns across countries, including variations in temperature, rainfall, and extreme weather conditions, indicating uneven climate vulnerability across the region.
